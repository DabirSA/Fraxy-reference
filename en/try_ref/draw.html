- FRAXY Try function (Draw) -
COLOR "COLOR",R,G,B
各命令で使用する色を設定する命令です。
"MESSAGE"、"EXMESSAGE"、"FLASH"でこの設定が使用されます。
書式"COLOR",int1,int2,int3
int1R (赤 0~255)
int2G (緑 0~255)
int3B (青 0~255)

DELEXMESSAGE "DELEXMESSAGE",ID,option
"EXMESSAGE"で作成したメッセージを削除する命令です。
間違ったIDを指定するとなんらかの問題が発生する場合があるので注意してください。
書式"DELEXMESSAGE",int1,int2
int1削除するメッセージのID
int2削除オプション
0
即座に消去
1
フェードアウト

EXMESSAGE "EXMESSAGE","message",align,basepos,pos_x,pos_y,time,option
画面上にメッセージ作成し表示する命令です。
文字色は"COLOR"、フォントは"FONT"で設定します。

この命令が成功した場合、変数(_stat)にメッセージのIDが返ります。
この命令が失敗した場合、変数(_stat)に -1 が返ります。
書式"EXMESSAGE","str1",int2,int3,int4,int5,int6,int7
"str1"メッセージ
int2文字揃えフラグ
0
左揃え
&1
中央揃え(X)
&2
右揃え
&4
中央揃え(Y)
&8
下揃え
int3基準となる位置
012
345
678
int4位置補正 X (ドット)
int5位置補正 Y (ドット)
int6表示時間 (フレーム単位)
>0
指定フレーム値の間表示
-1
永久に表示
int7表示オプション
0
標準
&1
点滅
&2
袋文字
&4
一文字ずつ表示
&8
フェードインさせる

FONT "FONT","fontname",size,style
"EXMESSAGE"で使用するフォントを設定する命令です。
フォント名には"MS ゴシック"や"MS 明朝"といったフォントの名称を指定します。
全ての環境で全てのフォントが揃っているとは限りませんので汎用性の高いフォントの使用を推奨します。
フォント名を省略した場合はユーザーがコンフィグで設定しているフォントが指定されます。
書式"FONT","str1",int2,int3
"str1"フォント名 [UserFont]
int2サイズ [ignore]
int3スタイル [ignore]
0
標準
&1
ボールド
&2
イタリック

MESSAGE "MESSAGE","message",R,G,B,option
左上のシステムメッセージにメッセージを表示する命令です。
色の指定を省略した場合は"COLOR"で指定されている色が使用されます。
書式"MESSAGE","str1",int2,int3,int4,int5
"str1"メッセージ
int2R 赤 (0~255) [R("COLOR")]
int3G 緑 (0~255) [G("COLOR")]
int4B 青 (0~255) [B("COLOR")]
int5表示オプション
0
標準
1
点滅

INTERFACE "INTERFACE",int1
プレイヤーのインターフェースの表示を変更する命令です。
書式"INTERFACE",int1
int1表示フラグ
-1
フェード無しで非表示にする
0
フェードアウトで非表示にする
1
フェードインで表示する
