- FRAXY Try function (Old) -
この命令群は過去に作成されたトライ等の互換性のために残されています。
通常は使用する必要ありません。
MOV "MOV",variable,value
変数に値を代入する命令です。
現在のバージョンでは以下のように記述できます。

var1 = int2
書式"MOV",var1,int2
var1対象変数
int2代入する値

ADD "ADD",variable,addvalue
変数に値を加算する命令です。
現在のバージョンでは以下のように記述できます。

var1 + int2
var1 += int2
var1 = var1 + int2
書式"ADD",var1,int2
var1対象変数
int2加算する値
