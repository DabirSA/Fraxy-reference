[C] Core

Core parts are important parts which form the heart of the game's action.

Every enemy will be built around one of these core parts.
Core type - Rolling Core, Rolling Core 2, Big Core, Big Core 2, Cancer Core, Small Core, Fighter Core, Fighter Core 2, Missile Core, Block Core, Mini Core, Bit Core, Deploy Core, Head Core,

(062) Deploy core
A special core part used to make multiple enemies appear at the same time.
Silently self-destructs as soon as it spawns, leaving its child parts behind.
All child parts that are not parented to a core part will also be destroyed.
