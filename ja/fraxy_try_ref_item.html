- FRAXY Try function (Item) -
ITEM "ITEM",itemtype,angle,distance
アイテムを生成する命令です。
生成位置はプレイヤーの位置を基準に方向と距離で指定します。
書式"ITEM",int1,int2,int3
int1アイテムの種類
-4
最上級アイテム
-3
上級アイテム
-2
中級アイテム
-1
最下級アイテム
0
通常のアイテム
1
"ITEMCONSTRUCT"で設定したアイテム
int2生成方向 (0~4095)
int3生成距離 (0~)

ITEM2 "ITEM2",itemtype,x,y
アイテムを生成する命令です。
生成位置は"POS"で設定されている位置を基準に相対座標(X,Y)で指定します。
書式"ITEM2",int1,int2,int3
int1アイテムの種類
-4
最上級アイテム
-3
上級アイテム
-2
中級アイテム
-1
最下級アイテム
0
通常のアイテム
1
"ITEMCONSTRUCT"で設定したアイテム
int2生成位置 X
int3生成位置 Y

ITEMCONSTRUCT "ITEMCONSTRUCT",itemnumber,itemID...
生成するアイテムの構成を変更する命令です。
アイテム数に設定した数だけint2~8にアイテムIDを設定してください。
書式"ITEMCONSTRUCT",int1,int2...
int1アイテム数 (1~7)
int2~8アイテムID (1~14)

ITEMSCROLL "ITEMSCROLL",scrollspeed,scrollspeedfade,scrollangle,scrollanglefade
アイテムをスクロールさせる命令です。
各パラメータは省略すると値を変更しません。
書式"ITEMSCROLL",int1,int2,int3,int4
int1スクロール速度 [ignore]
int2スクロール速度のフェード値 (1~) [ignore]
int3スクロール方向 (0~4095) [ignore]
int4スクロール方向のフェード値 (1~) [ignore]
