- FRAXY Try function (Timer) -
START "START"
タイマーのカウントを開始する命令です。
書式"START"

STOP "STOP"
タイマーのカウントを停止する命令です。
書式"STOP"

TIMER "TIMER",timervalue
タイマーを設定する命令です。
-1を指定するとタイマーを無効化します。
タイマーの値は変数(_time)で取得できます。
書式"TIMER",int1
int1設定値
1~
指定した時間(秒)をタイマーに設定する
-1
タイマーを無効化する

TIMERADD "TIMERADD",value
タイマーの時間に指定された時間を加算する命令です。
マイナスの値を指定すれば減算できます。
書式"TIMERADD",int1
int1加算する時間 (秒)

TIMEREND "TIMEREND",management,"labelname",scriptID
タイマーのカウント終了時の処理を設定する命令です。
この命令を実行しなかった場合はゲームオーバー処理になります。

goto/gosubの場合、どのスクリプトIDでジャンプするかを指定できます。
スクリプトIDを省略もしくは 0 を指定するとメインスクリプトが対象になります。
書式"TIMEREND",int1,"label2",int3
int1処理方法
0
ゲームオーバー
1
クリア
2
指定したラベルへジャンプする (goto)
3
指定したラベルへサブルーチンジャンプする (gosub)
"label2"ジャンプ先ラベル名
int3スクリプトID
